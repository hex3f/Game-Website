# 游戏官网 - 带用户热力图
主要功能是实现用户在注册账号后在用户所在位置地图上生成一个点

* 搭建教程：
1. 打开check文件夹的regcheck.php，在第93行填写百度地图的AK号码。
2. 打开config文件夹的config.php，填写极验GeeTest的ID和Key。
